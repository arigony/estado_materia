# Simulação Interativa — Estados da Matéria e Forças Intermoleculares

Este projeto apresenta uma **simulação interativa em HTML, CSS e JavaScript** sobre **estados da matéria** e **forças intermoleculares**.

O material foi desenvolvido para uso didático e permite visualizar, de forma simples e dinâmica, como a **temperatura**, a **organização das partículas**, a **energia cinética** e as **interações intermoleculares** se relacionam na passagem entre os estados **sólido**, **líquido** e **gasoso**.

## Objetivo

O objetivo desta simulação é ajudar estudantes a compreenderem, de maneira visual, que:

- no **estado sólido**, as partículas ficam mais próximas e organizadas
- no **estado líquido**, as partículas continuam próximas, mas com maior mobilidade
- no **estado gasoso**, as partículas ficam muito mais afastadas e com maior agitação
- as **forças intermoleculares** influenciam diretamente a coesão entre partículas e, portanto, o comportamento físico das substâncias

## Funcionalidades

A simulação inclui:

- seleção de diferentes **substâncias-modelo**
- ajuste de **temperatura relativa**
- mudança direta entre os estados:
  - **sólido**
  - **líquido**
  - **gasoso**
- visualização animada do movimento das partículas
- representação qualitativa de:
  - **forças de London**
  - **interações dipolo-dipolo**
  - **ligação de hidrogênio**
- indicadores de:
  - **energia cinética**
  - **organização**
  - **distância média entre partículas**
  - **intensidade relativa de atrações e agitação**

## Substâncias incluídas

A simulação trabalha com exemplos simples de diferentes perfis intermoleculares:

- **Argônio (Ar)** — forças de London
- **HCl** — London + dipolo-dipolo
- **Água (H₂O)** — London + dipolo-dipolo + ligação de hidrogênio
- **Amônia (NH₃)** — London + dipolo-dipolo + ligação de hidrogênio
- **CO₂** — predominância de London, por ser globalmente apolar

## Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript puro**
- **Canvas API** para a animação das partículas

## Como usar

1. Baixe o arquivo HTML do projeto.
2. Abra o arquivo em um navegador moderno.
3. Escolha uma substância no menu.
4. Ajuste a temperatura no controle deslizante.
5. Observe a mudança no arranjo das partículas e nas forças intermoleculares.
6. Use os botões de estado para alternar rapidamente entre sólido, líquido e gasoso.

## Aplicação didática

Este material pode ser usado em:

- aulas de **Química Geral**
- aulas de **Físico-Química**
- introdução às **forças intermoleculares**
- revisão de **mudanças de estado físico**
- atividades de comparação entre substâncias polares e apolares

## Sugestões de uso em sala de aula

- comparar uma substância com **ligação de hidrogênio** com outra que apresenta apenas **forças de London**
- discutir por que gases apresentam interações menos frequentes
- relacionar **temperatura** com **energia cinética média**
- pedir que os alunos expliquem por que a água apresenta comportamento diferente de substâncias apolares
- usar a simulação como ponto de partida para discutir:
  - ponto de ebulição
  - volatilidade
  - polaridade molecular
  - intensidade das forças intermoleculares

## Ideias de perguntas para os alunos

- Por que as partículas no sólido aparecem mais organizadas?
- O que muda quando aumentamos a temperatura?
- Por que a água apresenta ligação de hidrogênio e o CO₂ não?
- Em qual estado as forças intermoleculares parecem atuar com menor frequência?
- Como a polaridade influencia o tipo de interação entre moléculas?
